## Setup

#### Clone the repository
```shell
git clone https://github.com/k3rnel-dev/MacroZORTM
cd MacroZORTM
```

#### Install
```shell
chmod +x install.sh
sudo ./install.sh
```

## Usage
```shell
macroZORT 
```
